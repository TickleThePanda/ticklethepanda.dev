In an effort to keep fit and healthy, I keep track and analyse my fitness data. I have created a simple web service to keep track of my weight. See my health-vis repository on GitHub.

Collection methodology

This web service consists of endpoints to get and store data for weight. This data is stored in a Postgres database. This service and database is deployed to Heroku.
